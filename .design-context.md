# PitLife Design Context

## Product
- MonoGame ecosystem simulation for desktop players.
- Interface language: Italian.
- Audience: general PC players; controls must work with mouse and keyboard.

## Brand
- Personality: playful, naturalistic, readable.
- Direction: naturalist pixel art, selected by the project owner.
- Existing signals: illustrated ecosystem logo, pixel-art creatures and biomes.

## Accessibility
- Target WCAG AA contrast where applicable to a game UI.
- Keyboard navigation and visible focus are required.
- Primary controls use targets of at least 44x44 pixels.

- None.
